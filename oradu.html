CSS grid Web stranica

U ovom diplomskom radu prikazan je način rada CSS 3 komponente - CSS Grida, odnosno kako uz pomoć ovog sustava kreirati potpunu web stranicu. Kao primjer koristit ćemo oglednu web stranicu koja će biti kreirana kroz CSS Grid. Kroz ovaj rad bit će opisani glavni dijelovi i značajke CSS Grida, njegova svojstva kao što su stvaranje rešetke, dodavanje elemenata, pozicioniranje elemenata, grid staza, praznine, te mnogi primjeri mogućnosti uporabe Grida. Rad je podijeljen na dva dijela. Prvi dio prolazi kroz teoretski dio koji je nužan za poznavanje kako bi se mogao razumjetio drugi dio koji se odnosi na sami projekt kojeg sam napravio. Prvi dio sačinjen je od niza primjera na kojim se opisuju funkcionalnosti. Tek pred kraj, koristeći CSS Grid kreiramo često uporabljivane izglede stranica na internetu. Drugi dio je sama ogledna prezentacijska stranica koja ima za cilj predstaviti naučene vještine. Tu je opisana stranica, pojedini dijelovi i objašnjen kod koji stoji iza toga. Danas postoje mnogi načini frontend programiranja i mnoge razvijene tehnologije, frameworksi, ali cilj ovoga rada je prikazati kako se sa CSS 3 tehnologijom, u kojoj je sadržan CSS Grid, može postići jednako dobar rezultat uz mnoge intuitivne prednosti koje sa sobom nosi CSS Grid. Treba napomenuti da će se u ovom radu koristiti neki stručni izrazi na engleskom jeziku, što nije neuobičajeno s obzirom na to da nisu definirani formalno usvojeni prijevodi.

Šta je Grid?

Rešetka je križajući niz horizontalnih i vertikalnih linija – jednih koje definiraju stupce, a drugih redove.

Pozicioniranje?

Elemente možete postaviti na preciznu lokaciju na rešetki pomoću brojeva linija, imena ili ciljanjem područja rešetke (grid).

Kreiranje Grida?

HTML element postaje grid spremnik proglasujući display svojstvo kao grid ili inline-grid.

Podrška?

Chrome, Firefox, Safari i Edge podržavaju Grid!
