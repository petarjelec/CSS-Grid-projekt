BIO

Ja sam Petar Ban Jeleč, student 3. godine Računarstva na FSR-u. Za sebe kao studenta mogu ukratko reći: nikad najbolji, uvijek inovativan. Studentski dani su bili zanimljivi, ponekad manje ponekad više, ali uvijek vođeni trudom, radom i željom da se nauči nešto novo. Dosadašnji angažman u programerskom svijetu ostvarivao sam od kuće. Ovaj rad je jedan od projekata koje sam uspješno obavio. Neke od tehnologija u kojima imam iskustva:

CSS 3

Cascading Style Sheets

Poznavatelj CSS 3 tehnologije za prezentaciju web stranica. Radio par projekata kao frontendaš u ovoj tehnologiji. Sposoban za samostalan rad.

Bootstrap

Open-source framework

Svakodnevno koristi i Bootstrap. Bootstrap je besplatan i open-source front-end framework za izradu web stranica i web aplikacija

Vue.js

Javascript framework

Poznaje Vue.js i aktivno se educira u smjeru poboljšanja poznavanja ove tehnologije.Vue.js je open source JavaScript za izgradnju korisničkih sučelja
